Fyp

Arduino based obstacle avoidance robot

Welcome to the fyp Github page

This is the Git repository for Daniel Atkinson, a final yeal computer science undergraduate at Aberystwyth University

The following content will be related to his final year project hence the name "fyp"

Basic Idea

The base goal of this project is to design, build and program a small wheeled robot.
This robot is to independantly move around freely within its environment and avoid the various obstacles it may face

Background

I tend to experiment with electronic components on a regular basis. Due to this I try and make small working systems, originaly nothing more than a small timer or an audio amplifier.
Naturaly the progression from pure electronics was to move onto microcontrollers given my computer science background.
After making various projects using pic and arduino microcontrollers I decided to make an Arduino based robot.
This robot was just a chassis kit (magician), which included the frame, two wheels, two motors and a battery holder. From this all I had to add was an Arduino microcontroller and a sensor, for this I used a sharpe IR (infra-red) sensor, mounted inside the shassis pointing out of the front.
The code for this early 'prototype' was very simplistic and was along these sorts of lines: 

void loop(){
if(sensor_range < value)
	{
		motors.turn(random);
	}else
	{
		motors.forward(1);
	}
}
